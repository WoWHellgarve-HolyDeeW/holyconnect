HOLYCONNECT - START HERE

Most users only need 2 files:
1. HolyConnect-Run-First.bat
   Use once per SD card on a Windows PC before the first real Pi boot.
2. HolyConnect.bat
   Use for normal day-to-day USB access after the first-time setup is done.

Ignore the advanced\ folder unless you want manual control or the README explicitly tells you to use an advanced fallback.

Normal flow:
1. Put the official Pi-Star .zip or .img in pistar-image\
2. Insert the SD card into the Windows PC
3. Run HolyConnect-Run-First.bat
4. Put the SD card into the Pi and boot once
5. After the automatic reboot, connect the Pi to the PC by the USB DATA port
6. Run HolyConnect.bat

If you later copy only the windows\ folder to another PC, use windows\HolyConnect.bat on that PC.
