HOLYCONNECT - START HERE

Simplest path: double-click HolyConnect-Run-First.bat and click Yes if Windows asks for administrator rights.
The launcher auto-detects whether the SD card already has Pi-Star on it and opens the right helper.

For a new SD card:
1. Put the official Pi-Star .zip or .img in pistar-image\
2. Insert the SD card into the Windows PC
3. Double-click HolyConnect-Run-First.bat
4. Put the SD card into the Pi and boot once
5. After the automatic reboot, use windows\HolyConnect.bat

If the SD card already has Pi-Star on it:
1. Insert the SD card into the Windows PC
2. Double-click HolyConnect-Run-First.bat
3. Put the SD card into the Pi and boot once
4. After the automatic reboot, use windows\HolyConnect.bat
