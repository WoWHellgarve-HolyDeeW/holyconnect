HOLYCONNECT - ADVANCED HELPERS

Most users should ignore this folder.

Normal flow:
1. Run HolyConnect-Run-First.bat once per SD card.
2. After the first Pi boot is done, run HolyConnect.bat for normal USB use.

Use the files in this folder only if you want manual control:
- FlashPiStarSD.bat = force a full SD flash from an official Pi-Star image
- PreparePiStarSD.bat = force boot-partition prep on an already clean Pi-Star card
